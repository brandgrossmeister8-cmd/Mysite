# Персональный сайт-визитка

Современный персональный сайт-визитка с шахматной тематикой для Маргариты Осмаевой - бренд-гроссмейстера, консультанта по стратегическому маркетингу и системному продвижению.

## Описание

Одностраничный сайт с элегантным дизайном в шахматной стилистике. Сайт представляет профессиональную информацию, услуги и достижения в области маркетинга и продвижения бизнеса.

## Особенности дизайна

- Шахматная тематика с использованием шахматных фигур в оформлении
- Анимированная аватарка с вращающейся шахматной рамкой
- Адаптивный дизайн для всех устройств
- Элегантная цветовая палитра с использованием золотых акцентов (#d4af37)
- Плавные градиенты и тени для создания глубины

## Структура проекта

```
Mysite/
├── index.html          # Основная HTML страница
├── style.css           # Файл стилей
├── mysite/
│   ├── myfoto.jpg     # Фотография профиля
│   └── mybio.txt      # Текстовый файл с информацией
└── README.md          # Этот файл
```

## Технологии

- HTML5
- CSS3
- Адаптивная верстка (responsive design)
- CSS анимации и трансформации
- Шрифт Georgia для элегантного вида

## Разделы сайта

1. **Профиль** - Имя, должность и опыт работы
2. **Описание** - Краткая информация о деятельности
3. **Профессиональные установки** - Ключевые принципы работы
4. **Обо мне** - Достижения и регалии
5. **Мои услуги** - Перечень предоставляемых услуг
6. **Контакты** - Раздел для связи (готов к добавлению ссылок)

## Установка и запуск

1. Клонируйте или скачайте проект
2. Откройте файл [index.html](index.html) в браузере

Никаких дополнительных зависимостей или сборки не требуется - это статический сайт, готовый к использованию.

## Настройка

Для изменения контента отредактируйте:
- [index.html](index.html) - для изменения текста и структуры
- [style.css](style.css) - для изменения внешнего вида и цветов
- [mysite/myfoto.jpg](mysite/myfoto.jpg) - замените на свою фотографию

## Цветовая палитра

- Основной цвет фона: `#2c3e50` и `#34495e` (темно-синий градиент)
- Акцентный цвет: `#d4af37` (золотой)
- Текст: `#2c3e50` (заголовки), `#555` и `#444` (основной текст)
- Фон карточки: белый с градиентом до `#f5f5f5`

## Адаптивность

Сайт адаптирован для:
- Десктопных экранов (> 768px)
- Планшетов и мобильных устройств (≤ 768px)
